HTML

  • HTML stands for hypertext markup language and provides basic strcture to the webpage
  • h1, h2, h3, etc. elements are all headers, h1 being the biggest, and becoming smaller as you go. They are important for screen readers
  • Head element is all metadata, stuff the user will not see, other than the title.
  • Body element is all the actual content on the webpage that user sees and interacts with.
  • A img element has two parts: src="" which has the source of the image, being from the internet or a local file. and alt="" which holds the descripton of the imager for screen readers and search engines (very important)
  • There are two types of list. Unordered list (ul) (like this one) has bullet points. The other is an Ordererd List (ol) and has numbers. Every line in the list starts with li

CSS

  • CSS stands for cascade style sheets.
  • It is used to give certain styles, like color, size, placement, decorations, etc. to specefic elements of a html file.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • A border is the line between margin and padding.
  • You can define a class and give it to certain elements or sections of html code to give those things in the class a certain style.
  • You can group elements togther by putting them in sections

Git

  • Git is a version control program that works together with github to make it much easier to work with others and make sure to not break your website by using branches of the main site.
  • Some console commands to remember are: ls: stands for list. Displays files and directories
  • cd: Stands for change directory. Can be used to navigate directories in computer. Example: "cd c/Users/(user)/dowloads" will go to my downloads folder "cd /" will go to the very surface level of the pc. "cd .." will go one level up.
  • "mkdir (name)" will make a directory/folder with that name.
  • "touch (name)" will make a file with that name.
  • "start (filename)" will start a file as if you double clicked on it.
  • Now for actual git commands: git status: checks what branch we are currently on
  • git checkout (branch name): switches current working branch to the one typed in angled brackets
  • git pull origin main: pulls project from most current main branch.
  • git checkout -b (new branch): makes a new branch and switches current working branch to that one

Javascript

  • Javascript is the language a website uses for everything the user will interact with.
  • A variable in Javascript is declared using "var (variable name) = (variable value);" It is robust and can hold any value unlike Java.
